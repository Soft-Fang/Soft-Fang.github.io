简单认识Python

  • .print(5+3) 不需要“ ” 括起来的表示字符串（文本，要python输出的东西）

    而5+3为数字（整形）

  • 退出python环境方法：输入exit()或quit（）在记事本里输入python指令后，将文件命名为？？.py，然后在cmd中打开文件本地目录，再直接输入python 文件名.py 回车就可以运行其中指令

  • 特殊注释目前基本不需要，只有在python2版本需要这两个注释：
    1.#!/usr/bin/python3
    (表示指定python解释器路径)
    2.#–coding=utf-8
    (表示指定编码格式先版本python默认为utf-8,为的是编码方式支持中文)

  • BIF==built-in function(内置函数)
    input也是个BIF 功能：1.显示参数 2.要求用户输入一个字符串（数据），然后把数据返回给变量
    int也是BIF 功能：将后面框内的参数变为整形的变量

变量

用变量以前先赋值
变量名可以包括字母，数字，下划线，但变量名不能以数字开头。
字母可以大小写，但大小写意义不同
等号（=）是赋值的意思，左边名字，右边值，不可调换
变量可以取任何值，但作为程序员应取更专业的名字以便理解

字符串

5+8输出为13 “5”+“8”输出为“58”
如果需要出现 ‘ 或 “ 可以通关过在前面加上\ 例：Let’s go Let\ ‘s go

原始字符串

对于str='C:\now' 系统运行时会把\n当作转行 所以要用上原始字符串 即改为str=r'C:\now'
对于str=r'C:\now\123\123\' 系统会报错最后不能跟 \ 如果非要最后加上\那么还需要跟上一个空格
