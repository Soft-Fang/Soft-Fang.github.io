数据类型

符号常量

用标识符代表一个常量 成为符号常量

八进制整常数 以0开头作为前缀

十六进制整常数 以0X或0x作为前缀
123456789abcdef按顺序表示1~15

补充：内存中的二进制存储
一个字节byte=8位bit

数值是以补码表示的：

正数的补码:原码相同
负数的补码：将该位数绝对值的二进制形式按位取反再加1

例：求-10的补码
10的源码：00001010
取反：11110101
再加1，得-10的补码：11110110

实型数据

实型常量

实型也称为浮点型。实型常量也称为实数或浮点数。在C语言中，实数只采用十进制。它有两种形式：十进制小数形式，指数形式。

指数形式：a E(e) n(a为十进制数，n为十进制整数)

例：2.1E5（等于2.1*10^5）

实型变量

实型变量在内存中以指数形式存储

可分为：

  • float
  • double
  • long double

例：3.14159 小数部分以指数形式表示为

+ .314159 1

数组

数组的定义与使用方式

数组 是具有相同类型的数据的有序集合
数组中每个数据称：1.数组元素 2.下表变量
C语言中使用数组必须先定义

定义方式： 类型说明符 数组名[常量表达式]
例：int a [10]; 表示定义了一个整型数组，数组名为a，此数组有10个整型变量元素

  • 数组下标永远是从0开始的 上述10个元素便为0~9，而没有10
  • 数组的大小不能在运行时依赖变量定义
  • 数组大小不等于0 要用[ ]
  • int a[10]; 表示定义数组长度为10
    t=a[6]; 表示引用序号为6的元素（从0开始） 即t=7

数组的赋值

数组的初始化赋值

类型说明符 数组名[常量表达式]={值，值，…}
int a[10]=(0,1,2,3,4);

这样部分赋值 后五个元素的值都会是0

函数

认识函数

由主函数调用其他函数，其他函数也会互相调用。同一个函数可以被一个或多个函数调用任意多次

函数的两类：

1.无参数函数 void printstar( ) void表示没有返回值 （ ）里为空表示没有参数

2.有参数函数

形式参数和实际参数

  • 在有参函数小括号内的变量称为 形参
  • 在主调函数小括号中的变量成为 实参
    返回值用return（ ）返回

不同函数之间传递数据，可以使用的方法：

  • 参数：通过形式参数与实际参数
  • 返回值：用return语句返回计算结果
  • 全局变量：外部变量
  • 形参与实参类型相同或赋值兼容
  • 形参不会影响实参
  • 被调用函数定义出现在主调函数前，可以不加声明

函数的嵌套调用

在定义一个函数时，其内又包含另外一个函数的完整定义。
C语言不能嵌套定义，但可以嵌套调用

局部变量

在一个函数内部定义的变量是内部变量，它只在本函数范围内有效

全局变量

限制使用全局变量：

1.占据较多内存

2.降低程序清晰性

3.函数通用性降低

  • 在函数外定义的变量为全局（全程）变量
  • 全程变量可使用的范围为：其从其定义位置开始一下的所有语句都可使用
  • 主函数与定义函数内的同名变量意义不同，不会发生冲突

变量的存储类别

动态储存方式与静态储存方式：

通过变量值存在的时间（即生存期）进行变量区分。

  • 静态是指在程序开始时由系统分配固定存储空间

  • 动态是指在需要时再分配

    C语言中每个变量与函数有两个属性：数据类型和数据储存类别

auto

auto（自动）变量为函数中的局部变量，属于动态存储方式

  • auto a <=>auto int a <=> int a
  • static a<=>static int a 默认为整型

static

用static声明局部变量

  • 静态的局部变量：函数中的局部变量的值在调用结束后不消失而保留原值
  • 静态局部变量是在编译过程中赋初值的，即只赋值一次，在运行时已有初值
  • 静态局部变量在函数调用结束后仍然存在，但其他函数不能引用它

register变量

变量（静，动）是存放在内存中的
为提高效率，可以把局部变量的值通过用register声明后放在CPU的寄存器(寄存器数量有限)中，需要时可以直接取出，而不必到内存中存取 在寄存器中存取的效率更高

extern

用于声明外部变量，以扩展外部变量的作用域 把函数中的变量定义为全局变量

需要建立储存空间的声明称为定义性声明
不需要的称为引用性声明

  • 内部函数：在定义的函数前加上static 那么这个函数就不能被其他文件使用
  • 外部函数：在定义的函数前加上extern或什么也不加，则此函数可以被其他文件调用

指针

认识指针

内存区的每个字节有一个编号（地址）
直接访问如：a=6；
间接访问如：scanf(“%d”,&a);
指针是一个特殊的变量，是用来存放地址的

例：
int * i_pointer *此处的意义不是取值操作符而是定义后述变量为指针**

可以通过 i_pointer=&i; 可以认为输入值给 i 再把 i 的值放到该地址

  • 知道了一个变量的地址，就可以通过这个地址来访问这个变量，因此，把该地址称为该变量的“指针”

  • C语言中可以定义一类特殊变量，专门用来存放变量的地址，称为指针变量

  • 注意！！！
    指针变量的值（即指针变量中存放的值）是地址（即指针）。

    指针是一个地址，而指针变量是一个变量。

    定义指针变量时必须指定基类型

& 与 *

*称为取值操作符

& 称为取址操作符

数组与指针

  • 一个变量有地址，一个数组包含若干元素，每个数组元素都在内存中占用存储单元，它们都有相应的地址。
  • 指针变量可以指向变量，也可以指向数组元素
  • 数组元素的指针就是数组元素的地址

实参用数组名，形参用指针变量
